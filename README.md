# The Barrington School — Website Concept Review

Interactive preview of five homepage and Jerome Village location-page concepts for
[thebarringtonschool.com](https://www.thebarringtonschool.com), prepared by **Flywheel Advertising**.

## 🔗 Live preview

Once GitHub Pages is enabled, the review is available at:

> `https://<your-username>.github.io/<repo-name>/`

(Locally: open **`barrington-concept-preview.html`** in any browser.)

## What's here

| File | Description |
|------|-------------|
| `barrington-concept-preview.html` | The interactive preview — switch between all five concepts, page types, and device sizes |
| `Barrington-Concept-Review-Guide.pdf` | Navigation & usage guide (two creative directions + how to review) |
| `barrington-*-concept-v2 (*).html` | The five concepts as individual pages (the preview already embeds them) |
| `assets/` | Logos and web-optimized photography used by the concepts |
| `index.html` | Landing redirect to the preview (for GitHub Pages) |

## The two creative directions

- **Direction A — Editorial / Warm** — refined serif + ivory/champagne palette (steps away from the 2021 Brand Book).
- **Direction B — Brand-Faithful / Navy** — Raleway + the Brand Book navy palette (on-brand).

The preview keeps the homepage and location page in the **same direction** as you switch between them.

## How to navigate

Open the preview, then use the **page-type tabs**, **concept chips**, and **Desktop / Tablet / Mobile** toggle. The address bar updates (e.g. `#home-v24/tablet`) so you can share an exact view. See the PDF guide for details.
